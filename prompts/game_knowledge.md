### GAME MECHANICS

#### Gathering (Lumberjacking, Mining, Fishing, Foraging)
- **Click resource node → character auto-harvests in 1-second loops**
- Each loop: `random(0, weapon_level + skill_level) > node_difficulty` = success
- Success: gain 1 item + skill XP. Failure: retry next loop automatically.
- **Trees respawn in 25s**, rocks in 30s, fishing spots don't deplete.
- Higher-tier tools (Iron Axe > Bronze Axe) increase weapon_level bonus = higher success rate.
- You MUST have the right tool equipped: axe for trees, pickaxe for rocks, fishing pole for fish.
- **Foraging level requirements:** Blueberry Bush=1, Corn=5, Peach=5, Blue Lily=10, Tomato=15, Raspberry=15, Blackberry=20, Mustard=20, Paprika=25, Cumin=25. If your Foraging level is too low, clicking the plant does nothing.
- **Foraging XP comes from harvesting lower-level plants** (Blueberry Bushes at Foraging 1). Grind Blueberry Bushes to level up Foraging.

#### Crafting
- **Click crafting station** (anvil, cooking pot, alchemy bench, etc.) → opens recipe interface
- Select recipe → click craft. Requires skill level + ingredients. **No failure** — if you meet requirements, it succeeds.
- Crafting stations are inside buildings in Mudwich village.
- Key chains: Ore → Bar (smelting) → Weapon/Armor (smithing). Raw fish → Cooked fish (cooking). Ingredients → Potions (alchemy).

#### Combat
- **Click mob → character auto-walks and auto-attacks**. Combat is passive — no dodging or active abilities needed.
- Damage = function of (attack skill + equipment bonuses) vs (defender's defense + armor). All hits connect, damage varies from 0 to max.
- Attack style determines which skills gain XP: Hack=6 (Str+Def), Chop=7 (Acc+Def), Defensive=3 (Def only).
- **Wait 5-6s after clicking a mob** — don't re-click during combat (interrupts the attack loop). Keep waits short to minimize idle time.

#### Mob Drop Rates
- Quest items (tomato, strawberry, string, magic bead): **~5-10% per kill**
- Expected kills for 1 quest item drop: 10-20 kills of the right mob type
- Expected kills for full Scavenger set (2 tomatoes + 2 strawberries + 1 string): **50-100 kills total** across multiple sessions
- Tomatoes/strawberries drop from: Goblins, Cactus, Ogres (any of these work)
- String drops from: Goblins, Spiders
- **If you kill 15+ mobs of the same type with 0 quest item drops, switch to a different mob type or quest.** RNG streaks are normal — diversify rather than persist.
- Scavenger quest is a long-term objective — do NOT try to complete it in one grinding session. Collect drops opportunistically while doing other quests and leveling.

#### Equipment Requirements
- Iron Axe: Strength 10
- Higher-tier weapons scale similarly — check skill requirements before equipping.
- Equipment comes from quest rewards, mob drops, crafting, and shops.

#### Shops
- **Start Shop** (Clerk): Arrows (5g), Flasks (100g), Burgers (450g)
- **Forester**: Bronze Axe (1000g), Iron Axe (5000g) — buys logs
- **Miner**: Ores — buys ores
- **Sorcerer**: Staves (magic weapons)
- Shops unlock via quest completion (Foresting → Forester shop, Miner's Quest → Miner shop)

### NAVIGATION TIPS

- **Long routes (>15 tiles)**: Use `__navigateTo(x, y)` — it auto-chains short hops. No need to manually break paths into steps.
- **Short routes (<15 tiles)**: Use `__moveTo(x, y)`. If it returns "No path found", fall back to `__navigateTo`.
- **Warp combat cooldown**: The server blocks warping for 10 seconds after combat ends. If `__safeWarp` fails, call `__clearCombatState()`, wait 10+ seconds, then retry.
- **After death**: Respawn is at (328, 892). Click RESPAWN button, then use `__safeWarp(0)` to return to Mudwich.
- **Village buildings**: Buildings block pathfinding. Walk around them or use door tiles to enter.
- **Bridge crossing** (Mudwich to Snek area): `__navigateTo(230, 160)` handles the bridge automatically.
